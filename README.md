# MARVEL_GNN

Python port of MARVEL (Furtenbacher, Császár & Tennyson 2007; Furtenbacher & Császár 2012) plus a graph neural network for uncertainty calibration, outlier detection, orphan-node linkage, and quantum-number label-correction triage.

No source ported yet — `src/marvel_gnn/core/` and `src/marvel_gnn/gnn/` are currently empty scaffolding.
